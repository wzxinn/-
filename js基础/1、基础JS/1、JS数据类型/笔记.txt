JS他的核心叫做ECMAScript  -> ES

ES给定义了5种的简单的数据类型
和一种复杂的数据类型

简单的

Undefined = 未定义
没有给变量定义任何的数据类型

Null = 空

Boolean = 布尔值/真的或者假的

Number = 数字

String = 字符串
' ' " "

复杂的
Object = 对象

typeof 这个方法 可以帮助咱们看变量到底是什么数据类型

JS的报错类型

语法错误 - 就是完全不执行了 写的所有的JS都失效了

运行错误 - 就是之前都执行 
就是报错的那一句之后 和 那一句都不执行了
